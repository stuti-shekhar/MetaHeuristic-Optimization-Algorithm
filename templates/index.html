About Genetic Algorithm

Genetic Algorithm is the soft computing technique used to enhance the accuracy of our predictor

A genetic algorithm is a search heuristic that is inspired by Charles Darwin's theory of natural evolution This algorithm reflects the process of natural selection where the fittest individuals are selected for reproduction in order to produce offspring of the next generation.

Selection - This phase of the genetic algorithm selects individual solutions from the current population to produce a new generation of the population by applying the crossover between the selected individuals. In our project, we have used elitism selection strategy.

Elitism - In this strategy, one parent is selected form top k individuals based on their fitness value and their corresponding partner is selected from the other part of the population.

Pre-Processing of the Training Set

The dataset we used is obtained from Kaggle. In the pre-processing phase the missing values are handled and one-hot encoding is done on specific features.

  • This obtained dataset is split into training and test data. Features are eliminated as well as combined so as to reduce the dimensionality.
  • Missing values in the database are appropriately handled by replacing them with the most commonly occurring value in that feature.

The model predicts the class label and the same is displayed to the user via the web interface. This ends the prediction phase and the user knows which class the student belongs to.

Decision Tree

After the preprocessing, initial population of decision trees was constructed.

Single Point

Crossover Technique - I

In this crossover a single internal node is selected. The selected node should be a feature node in both of the parents. After a node is selected, the sub-tree of the two parent chromosome starting from that node is exchanged between the two chromosomes to produce two offspring chromosomes.

Dominance Crossover

Crossover Technique - II

In this crossover, a single internal node is selected. The selected node should be a feature node in both of the parents. After a node is selected, the correlation between feature of all the nodes in the sub-tree with attribute G3 of two trees is compared, If the absolute value of the correlation of one tree is greater than another tree the node with the lower value is replaced by a node with higher correlation value.
